About drasyl

drasyl ([d Åazy:l]) is a lightweight, software-defined overlay networking solution built on a fast and secure peer-to-peer protocol.

It enables you to seamlessly interconnect all your devices in your organization, team, or home, regardless of the network barriers commonly present between physical networks. Unlike traditional VPNs, drasyl builds direct peer-to-peer connections between all devices, creating a mesh overlay that efficiently utilizes the underlying physical network.

No proprietary, centralized network controller is required, as overlay configurations can be distributed using any standard HTTP server or fully decentralized with local copies on each device. You retain full control over network membership and which devices are permitted to communicate, following a zero-trust model with all traffic end-to-end encrypted and all devices authenticated.
