Install drasyl CLI via Homebrew

On November 20, 2020 in CLI

Homebrew is “The Missing Package Manager for macOS (or Linux)”. Therefore, it provides a very comfortable way to install/update the drasyl CLI.

To install drasyl via Homebrew, you need this command:

brew install drasyl-overlay/tap/drasyl

After installation is completed, the drasyl command should be available. Type drasyl help to get an overview of all available utilities.
