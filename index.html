drasyl

Next Generation Overlay Network

What it is

✻

drasyl is an open-source (under development), general-purpose overlay network that is concurrent, resilient, flexible, automated, and presents itself to the user as a transparent system that offers suitable discovery and awareness methods, particularly with a focus on smart city and IoT devices. Nevertheless, drasyl is not limited to smart city and IoT but is intended for universal use in all decentralized Java-based projects.

Aspired goals

✻

drasyl should offer many functions to quickly and flexibly create robust overlay networks for any application, which are not limited to a local network but can be connected agilely worldwide.

Dockerized

Each drasyl node can be operated as a virtualized container. We offer ready-to-use container images for this purpose. Together with Docker Swarm or Kubernetes, you can quickly create clusters for your overlay network.

Smart City Support

drasyl enables sensors and IoT devices to be operated outside a LAN/VLAN or VPN and is specially optimized for this application area.

Open Source

drasyl is a real open-source application and licensed under LGPL v3 which means that it can even be used in commercial software.

Easy to use

drasyl is designed as a ready-to-use application so that it appears to the developer as a transparent system. It never was easier before to build a worldwide overlay network.
